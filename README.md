# 📄 Personal Resume Website

A modern and responsive resume website built to showcase my professional experience, technical skills, projects, and achievements as a Full-Stack Developer.

## 🚀 Features

* Responsive design for desktop, tablet, and mobile devices
* Professional resume and career overview
* Skills and technology stack section
* Project portfolio showcase
* Contact information and social links
* Fast performance and optimized user experience

## 🛠️ Tech Stack

* framer motion
* React
* TypeScript
* Tailwind CSS
* Resend
* Next.js


## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/Info-Bash/bash-resume.git
```

Navigate to the project directory:

```bash
cd bash-resume
```

Install dependencies:

```bash
pnpm install
```

Start the development server:

```bash
pnpm dev
```

Open your browser and visit:

```text
http://localhost:3000
```

## 🏗️ Build for Production

```bash
pnpm build
pnpm start
```
